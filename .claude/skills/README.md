# WJIH Claude Skills

This folder contains project-specific Claude skills for the Wadi Jeddah Innovation Hub website.

* `wjih-brand-system`: Use before visual/UI changes to enforce brand colors, page themes, spacing consistency, and token reuse.
* `wjih-page-builder`: Use when building or editing pages, sections, shared page components, routes, and WhatsApp CTA behavior.
* `wjih-responsive-qa`: Use before merging UI work to check desktop/mobile behavior, navbar scroll behavior, RTL/LTR, CTAs, and build readiness.
* `wjih-i18n-content`: Use when editing Arabic/English content, translations, language keys, labels, or content structure.
