# CLAUDE.md

Claude should help with WJIH website UI, components, layout, responsive design, and content-to-section transformation while staying inside the approved MVP scope.

## Project Guardrails

* Do not build backend, dashboard, auth, database, CMS, or booking logic now.
* Do not invent new brand colors.
* Do not remove Arabic/English support.
* Do not remove RTL support for Arabic or LTR support for English.
* Do not hardcode long page content randomly inside JSX.
* Keep CTA/contact behavior routed to WhatsApp for the MVP.
* Keep changes focused on the requested files and task owner.

## Available Claude Skills

* `wjih-brand-system`: use before visual/UI changes.
* `wjih-page-builder`: use when building or editing pages.
* `wjih-responsive-qa`: use before merging UI work.
* `wjih-i18n-content`: use when editing Arabic/English content.
